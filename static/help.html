Alexa Web

This is a browser based interface to Alexa, Amazon's virtual assistant which poweres the Echo.
Alexa is not the same as an echo, echo has a number of additional features such as music playback and hotword detection that are not (yet) implemented here.
THIS IS NOT AN OFFICIAL AMAZON SERVICE IT IS AN INTERFACE TO THEIR API BUILT BY A INDEPENDANT 3RD PARTY

The interface is fairly simple, a coloured circle on the centre of the page indicates the state, a key to these colors is given below.
Once the page has loaded and you see a white circle it is ready to use, you click (or touch) the circle and hold down then speak your command eg "What time is it in London" then release, Alexa will process you command and then reply to you in speech when the circle is green. If you see a brief red circle there was an error, try your request again. If Alexa doesn't understand the request she will say so.
If you want to logout then click the ☒ symbol in the top right corner.

Absolutly no warrenty is given this site is built as a interesting demo it might go away without warning, I don't keep any details of your requests I simply store a cookie to reference your Amazon login token, this is removed when you logout. You should also understand Amazon's Privacy Policy with regards to Alexa when using this service as I am merely providing a gateway.
